## 1、强调文本

1. **加粗** ： `**加粗**` 或 `__加粗__`
2. *倾斜*： `*倾斜*` 或 `_倾斜_`
3. ~~删除~~： `~~删除~~`  
4. ***加粗且倾斜*** ： `***加粗且倾斜***`

## 2、表格

> 使用 | 来分隔单元格，使用 - 来分隔标题行和其他行  
依次是左对齐 、 右对齐 、居中对齐

| 左对齐 | 右对齐 | 居中对齐 |
| :--- | ---: | :---: |
| 1 | 2 | 3 |
| 4 | 5 | 6 |
